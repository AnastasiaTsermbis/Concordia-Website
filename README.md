This project was uploaded from here: https://github.com/YahiaAbouelnasr/Soen287_Project. This is a project comprised of 4 people. This project is a booking system for concordia. 

https://crbs-b358b.web.app is the website and has been deployed. Alternatively, you may deploy it yourselves by downloading the code and enter the code: %firebase deploy

You may create an account, and then login with it. Since this is a "beginner project", you may login with a madeup email. It will be saved on firebase, which handles the auth.
There will be two kinds of accounts, Student and Admin. One can book, the other manages bookings.

Upon login, you should browse through resources and attempt to book them, this was my part, both on the student and admin side.
